# 「时光回忆录」运营推广方案

## 1. 产品命名与Slogan

**正式名称**：时光回忆录  
**英文**：TimeMemoir

**Slogan选项**：
1. "不用打字，会说话就能写回忆录"（主打卖点）
2. "帮妈妈留住一生的故事"（情感牌）
3. "每一句话，都是给未来的礼物"（温情牌）

**推荐**：Slogan 1 + 2 组合使用

**私域**：
- 老年大学合作：免费讲座

## 5. 内容运营策略

### 5.1 用户故事专栏

**"100位母亲的故事"**
- 每周发布1位用户的回忆录节选
- 获得授权后，制作成短视频
- 激励用户：被选中的故事，免费升级精装版

## 6. 裂变传播设计

### 6.1 分享机制

**生成精美海报**：
- 老人生成回忆录后，可生成"故事卡片"
- 卡片包含：老人照片+金句+二维码
- 子女分享到朋友圈，自然传播

**家庭协作功能**：
- 子女可以补充照片和说明
- 多子女一起完善父母的回忆录
- 邀请机制：邀请家人加入协作

### 6.2 口碑激励

- 完成回忆录 → 奖励"故事大使"徽章
- 分享海报 → 获得下章节折扣
- 邀请好友 → 双方各得1主题
