# 「时光回忆录」UI/UX 设计方案

> 设计原则：「温暖、简洁、无障碍」

---

## 1. 首页 - 开始回忆

```
┌─────────────────────────────┐
│                             │
│     🌅 时光回忆录            │
│                             │
│  "帮妈妈记录一生的故事"       │
│                             │
│    ┌─────────────────┐      │
│    │  ▶️ 开始回忆    │      │
│    └─────────────────┘      │
│                             │
│    ┌─────────────────┐      │
│    │ 📖 继续上次的    │      │
│    └─────────────────┘      │
│                             │
│    [📚 我的回忆录]          │
│                             │
└─────────────────────────────┘
```

**设计要点**：
- 背景：暖黄色渐变（夕阳/时光感）
- 字体：首页标题 36sp，按钮文字 28sp
- 主按钮：圆角16dp，阴影效果
- 无障碍：点击区域 72x72dp

## 7. 无障碍设计规范

| 项目 | 规范 | 说明 |
|------|------|------|
| 字体大小 | 最小24sp，标题32-48sp | 老花眼友好 |
| 行高 | 1.5倍 | 阅读舒适 |
| 对比度 | >4.5:1 | WCAG AA标准 |
| 点击区域 | >56x56dp | 手指粗点也能点中 |
| 颜色含义 | 不单独依赖颜色 | 红绿色盲友好 |
| 语音反馈 | 每步操作都有语音确认 | 不识字也懂 |
| 震动反馈 | 录音开始/结束震动提示 | 听障友好 |

## 8. 配色方案

**主色调**：温暖夕阳
- 主色：#FF8C42（温暖橙）
- 辅色：#6A994E（生命绿）
- 背景：#FFF8F0（米白）
- 文字：#2D3436（深灰）
- 提示：#74B9FF（天蓝）

**情感传递**：温暖、怀旧、希望

---

## 9. 交互动效

### 录音按钮
- 默认：呼吸动画（轻微缩放）
- 按住：波纹扩散 + 震动反馈
- 录音中：声波可视化

### 页面转场
- 主题选择 → 提问页：卡片展开
- 提问 → 追问：从右滑入
- 完成章节： confetti 庆祝效果

### 语音播放
- 文字高亮同步
- 进度条显示
- 可调节语速（0.8x/1.0x/1.2x）
