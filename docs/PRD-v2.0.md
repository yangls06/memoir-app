# 「时光回忆录」PRD V2.0

**一句话价值主张**：不用打字，会说话就能为妈妈写回忆录

---

## 1. 产品定位

### 目标用户
- **主要**：60-75岁城市退休老人（有一定人生故事，会用智能手机基础功能）
- **次要**：40-55岁子女（为父母购买/协助使用）

### 用户画像

**王阿姨 - 典型用户**
- 68岁，退休教师
- 会微信语音，不会打字
- 眼睛老花，需要大字体
- 有很多故事想讲给孙辈听
- 痛点：想写回忆录但无从下手

**小李 - 子女用户**
- 35岁，产品经理
- 想帮妈妈记录一生但没时间
- 愿意付费¥300-500买孝心

## 2. 核心功能流程

```
【开始】
  ↓
【选择主题】童年/青春/工作/爱情/育儿/老年
  ↓
【AI语音提问】"王阿姨，说说您小时候最难忘的一次过年？"
  ↓
【老人语音回答】（自动转文字）
  ↓
【AI追问】基于5W1H
  ├─ "当时有哪些亲戚在场？" (Who)
  ├─ "年夜饭吃了什么特别的？" (What)
  ├─ "那是哪一年？" (When)
  ├─ "在老家还是城里？" (Where)
  ├─ "为什么那次过年最难忘？" (Why)
  ├─ "你是怎么帮妈妈准备年货的？" (How)
  ├─ "当时心情怎么样？" (心情)
  └─ "这件事对你后来有什么影响？" (思考)
  ↓
【确认/补充】老人确认或补充内容
  ↓
【保存章节】生成该主题章节
  ↓
【时间线整理】可视化人生地图
  ↓
【AI润色】白描版本
  ↓
【选择作家风格】汪曾祺/张爱玲/余华/杨绛/史铁生/白描
  ↓
【生成回忆录】导出PDF/EPUB/实体书
```

## 3. 功能详解

### 3.1 AI引导问题

基于人生阶段设计主题：
- 👶 童年时光 - 家乡、亲人、学校
- 💑 青春岁月 - 初恋、理想、朋友
- 💼 工作经历 - 职业生涯、同事、成就
- 👨‍👩‍👧 家庭生活 - 婚姻、育儿、家庭
- 🌟 人生感悟 - 智慧、遗憾、寄语

### 3.2 5W1H追问逻辑

**触发条件**：
- 回答少于50字 → 自动触发追问
- 关键词缺失（无时间/地点/人物）→ 针对性追问
- 情感平淡 → 追问心情维度

**追问策略**：
1. 首次回答 → AI提取关键要素
2. 缺失要素补全 → 针对性提问
3. 深度挖掘 → 情感+意义层面

### 3.3 时间线整理

AI自动识别回答中的时间信息，构建人生时间线：

```
人生时间轴
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1950  1968   1980   1995   2010   今
出生  高考   结婚   生子   退休   
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 3.4 AI润色 - 白描版本

处理流程：
```
原始口语 → 语音识别 → 去气口/重复 → 补全省略 → 白描文本
```

润色规则：
1. 去口语化：删除"那个""就是""嗯"等 filler words
2. 补全省略：补充主语、连接词
3. 修正时序：调整倒叙为顺叙
4. 保留口语特色：不改成书面语，保持老人语气

### 3.5 作家风格改写

可选风格：

| 风格 | 特点 | 适合内容 |
|------|------|----------|
| **汪曾祺式** | 平淡自然、市井气息、美食人文 | 日常生活、故乡回忆 |
| **张爱玲式** | 细腻敏感、比喻精妙、苍凉感 | 爱情、青春、女性成长 |
| **余华式** | 冷峻克制、黑色幽默、时代感 | 苦难、奋斗、历史变迁 |
| **杨绛式** | 温润平和、知性优雅、人生感悟 | 晚年回忆、家庭温情 |
| **史铁生式** | 哲思深邃、生命思考、内省 | 疾病、生死、人生意义 |
| **白描版** | 保留原貌、去口语化、简洁 | 追求真实的用户 |

## 4. UI/UX 规范

### 无障碍设计

| 项目 | 规范 | 说明 |
|------|------|------|
| 字体大小 | 最小24sp，标题32-48sp | 老花眼友好 |
| 对比度 | >4.5:1 | WCAG AA标准 |
| 点击区域 | >56x56dp | 手指粗点也能点中 |
| 语音反馈 | 每步操作都有语音确认 | 不识字也懂 |

### 配色方案

- 主色：#FF8C42（温暖橙）
- 辅色：#6A994E（生命绿）
- 背景：#FFF8F0（米白）
- 文字：#2D3436（深灰）
