# 时光回忆录 - TimeMemoir

> "不用打字，会说话就能为妈妈写回忆录"

一款帮助老年人用语音记录人生故事的 AI 应用。

## 核心功能

1. **AI引导问题** - 基于人生主题的智能提问
2. **语音朗读** - 问题语音播报，方便不识字的老人
3. **语音回答** - 老人说话，AI转文字
4. **5W1H追问** - AI基于缺失维度追问细节
5. **时间线整理** - 可视化人生地图
6. **AI润色** - 去除口语化，生成白描版本
7. **作家风格** - 6种文学风格改写
