# HEARTBEAT.md - Ace 进化系统检查清单

每次心跳检查以下事项：

## 1. 自我进化系统状态
- [ ] 检查 memory/errors/ 是否有新增错误
- [ ] 检查 memory/telemetry/ 任务执行情况
- [ ] 检查是否有需要关注的问题模式（同类错误≥3次）

## 2. 技能进化触发条件
- [ ] 检查是否有技能使用≥10次，需要自评估
- [ ] 检查是否有高频任务可以提炼为 SKILL.md

## 3. 记忆整理
- [ ] 检查 memory/ 目录是否需要归档
- [ ] 检查 daily notes 是否需要汇总到 MEMORY.md

## 4. 主动优化建议
如果发现以下情况，主动向用户汇报：
- 同类错误发生≥3次
- 某技能成功率<90%
- 发现可以自动化的重复任务
